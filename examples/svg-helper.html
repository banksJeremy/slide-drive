This document includes one or more embedded SVG documents, defining several custom fonts. These fonts are incomplete: they only define glyphs some characters. There will be multiple definitions of some fonts, each containing a different set of glyphs.

All of the loaded font data is persisted in the DOM in JSON. If the document is saved, reloaded, and more fonts are loaded, it should work just fine.

Each of the following paragraphs should be using a distinct font or style. Each falls back to monospace if a glyph is unavailable.
